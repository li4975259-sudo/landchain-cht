## 二、酱香型白酒基础知识

### 2.1 什么是酱香型白酒

酱香型白酒是中国白酒四大基本香型之一（另三种为浓香型、清香型、米香型），以**贵州茅台镇**为核心产区，因具有类似酱油复合香气而得名。

**国家标准定义（GB/T 26760）：**  
以高粱、小麦、水等为原料，采用高温大曲为糖化发酵剂，经固态发酵、蒸馏、贮存、勾兑而成，具有酱香突出、幽雅细腻、酒体醇厚、回味悠长、空杯留香持久特点的白酒。

**核心产区：**  
贵州省遵义市仁怀市茅台镇，地处赤水河下游，独特的微气候、水质、土壤与微生物群落，造就了不可复制的酱酒风土。

### 2.2 酱香酒的工艺等级

按原料与工艺投入，酱酒可分为以下等级（由高到低）：

| 等级 | 说明 | 特点 |
|------|------|------|
| **坤沙（大曲酱香）** | 整粒高粱为主，完整"12987"工艺 | 品质最高，陈香足，层次丰富 |
| **碎沙** | 部分高粱破碎，发酵周期缩短 | 口感较薄，成本较低 |
| **翻沙** | 用坤沙酒糟加新粮二次发酵 | 带有前次酒体风格 |
| **串香** | 食用酒精与酱酒香醅串蒸 | 非纯粮固态，品质最低 |

**怀仁集团旗下石荣霄、周秉衡等核心品牌均坚持坤沙（大曲酱香）工艺**，这也是"百元真坤沙"等核心卖点的品质基础。

### 2.3 "12987"酿造密码

酱香型白酒传统酿造遵循**"12987"**工艺密码：

- **1**：一年生产周期（端午制曲、重阳下沙，历经四季）
- **2**：两次投料（下沙、糙沙）
- **9**：九次蒸煮
- **8**：八次发酵
- **7**：七次取酒（分轮次，各轮次酒体风格不同）

基酒需**至少贮存3-5年**，优质产品往往加入更长时间的老酒勾调，方得成品。

### 2.4 酱香酒的感官特征

**色**：微黄透明，陈年老酒色泽更深  
**香**：酱香突出，辅以焦香、糊香、花蜜香等复合香气  
**味**：入口醇厚，细品有酸、甜、苦、咸、鲜五味协调  
**空杯**：饮后杯壁留香持久，优质酱酒空杯香可达24小时以上  
**体感**：适量饮用后体感舒适，醒酒较快（因人而异）

### 2.5 酱酒行业现状（2024-2026）

- 酱酒经历2020-2021年爆发期后进入**品质化、品牌化、平价化**新阶段
- 消费者更加理性，追求"真产地、真工艺、真实惠"
- 直播电商、私域运营成为酱酒销售重要渠道
- 茅台镇2000余家酒企中，具备历史底蕴、品质认证、供应链能力的企业更具竞争力
- **百元真坤沙**成为口粮酱酒新赛道，打破"好酱酒必贵"的消费认知

## 三、茅台镇酱酒历史与三茅文化

### 3.1 茅台酒的前世今生

今日之**贵州茅台酒**，其历史渊源可追溯至清代至民国时期茅台镇上的多家私营烧坊。1951-1953年，仁怀县人民政府以赎买、没收、接管方式，将三大烧坊国有化，合并成立**贵州省茅台酒厂**。

三大烧坊即民间所称的**"三茅"**：

| 烧坊 | 俗称 | 主要创始人 | 创立时间 |
|------|------|------------|----------|
| 成义烧坊 | 华茅 | 华联辉 | 清同治年间 |
| 荣和烧坊 | 王茅 | 王泽履（王立夫） | 1879年（荣太和） |
| 恒兴酒厂 | 赖茅 | 赖永初（接衡昌） | 1929年（衡昌） |

1915年，**荣和茅酒**与**华茅**以"中国贵州茅台酒"名义联合参加美国旧金山**巴拿马万国博览会**，获金奖，茅台酒由此蜚声国际。

### 3.2 石荣霄与荣太和烧坊

**石荣霄**（原名王映奎，1855-1925）是王茅（荣和烧坊）历史上的核心人物之一。

**生平要点：**
- 1855年生于茅台村，王家世代行医，有朝廷御医记载
- 8岁过继石姓富绅，改名石荣霄
- 1879年与孙全太、王立夫（天和盐号）合股开设**荣太和烧坊**
- 1915年荣太和更名**荣和烧坊**，所酿之酒获巴拿马金奖
- 1918年还宗王姓，后人冠以王姓
- 1951-1952年荣和烧坊被国有化并入茅台酒厂，石荣霄后人携家传秘曲归隐

**荣霄秘曲（荣霄散）：**  
石荣霄家族传承的制曲秘方，将中医药草本精华融入酒曲，是石荣霄品牌今日复兴的核心工艺资产。

### 3.3 周秉衡与衡昌烧坊

**周秉衡**（1887-？）是赖茅（恒兴酒厂）历史上的创始人。

**生平要点：**
- 贵阳人，青少年于贵阳书院求学
- 19岁在四川叙永结婚经商，后加入同盟会
- 1927年春与二舅余德盛赴茅台镇考察，背运叙永高粱试验酿酒
- **1929年**在茅台镇买地40亩，创办**衡昌烧坊**（衡昌酒房），建17个酒窖
- 1938年赖永初入股6万大洋，成立大兴实业公司，赖永初任经理，周秉衡任副经理
- **1941年**赖永初收购周秉衡股份，衡昌烧坊更名**恒兴酒厂**，产品改称**赖茅**
- 1951年后恒兴酒厂并入茅台酒厂，周秉衡后人于1986年前后恢复酿酒

**历史地位：**  
周秉衡与石荣霄分属茅台镇两大酿酒传奇谱系——石荣霄代表**王茅（荣和）**一脉，周秉衡代表**赖茅（衡昌/恒兴）**一脉，二者共同构成中国酱酒文化的半壁江山。

### 3.4 三茅与怀仁品牌矩阵的关系

怀仁产业集团以**三域增长**为平台能力，以**石荣霄（王茅谱系）**为核心酱酒品牌，以**周秉衡（赖茅谱系）**为重要历史IP与产品线延伸，形成"两大茅系创始人背书"的品牌叙事：

- **石荣霄**：1879年荣太和烧坊，王茅第一代创始人，146年传承
- **周秉衡**：1929年衡昌烧坊，赖茅前身创始人，近百年传承

这一布局使怀仁集团在酱酒历史文化叙事上覆盖**王茅 + 赖茅**双主线，区别于仅强调单一历史渊源的竞品。

## 七、酿造工艺与品质认证

### 7.1 制曲工艺

**时间**：端午前后（农历五月）  
**原料**：优质小麦，粉碎程度适中  
**流程**：
1. 润粮：小麦吸收适量水分
2. 粉碎：粗麦粉，保留部分整粒
3. 拌料：加母曲、水，搅拌均匀
4. 踩曲/机械制曲：制成曲块
5. 入仓发酵：高温堆积，温度达60-65℃
6. 贮存：曲块陈化至少3-6个月

**石荣霄特色 — 荣霄秘曲：**  
融入王家世代行医的草本精华，酒曲独特，是"茅系秘曲"的核心秘密。

### 7.2 下沙与糙沙

**下沙**（重阳前后，农历九月）：
- 整粒红缨子高粱80% + 破碎20%
- 润粮24小时以上
- 蒸粮糊化，摊凉加曲
- 堆积发酵4-5天，入窖30-40天

**糙沙**（下沙后约一个月）：
- 再次投粮，工艺类似下沙
- 完成两次投料，进入轮次取酒

### 7.3 七轮次取酒

| 轮次 | 时间 | 酒体特点 | 用途 |
|------|------|----------|------|
| 一轮次 | 下沙后 | 粮香重，度数高 | 基础酒 |
| 二轮次 | | 粮香、焦香 | 基础酒 |
| 三轮次 | | 酱香初显 | 基础酒 |
| 四轮次 | | 酱香明显 | 骨架酒 |
| 五轮次 | | 酱香突出 | 骨架酒 |
| 六轮次 | | 酱香、焦香、糊香 | 调味酒 |
| 七轮次 | | 焦糊香重，度数低 | 调味酒 |

### 7.4 贮存与勾调

**基酒贮存**：
- 分轮次、分等级陶坛贮存
- 至少3年，优质产品5年以上
- 贮存期间定期品评、合并

**勾调**：
- 以不同轮次、不同年份基酒组合
- 加入老酒提升陈香与层次
- 小批量、手工勾调，确保每批稳定

**石荣霄勾调标准**：  
每瓶加入窖藏数十年老酒精心勾调，在上百种基酒中寻找微妙平衡。

怀仁集团旗下产品通过多重品质保障：

1. **生产认证**：大曲酱香型白酒、纯粮固态发酵、酒类安全品质

## 八、产品品鉴与饮用指南

### 8.1 专业品鉴步骤

**观色**：  
倒酒入杯，观察色泽。优质酱酒微黄透明，无悬浮物，挂杯均匀。

**闻香**：  
- 空杯闻香：倒酒前闻空杯  
- 初闻：倒酒后立即闻，感受主体香  
- 深闻：晃动酒杯，感受复合香  
- 空杯留香：饮后闻杯，优质酱酒留香持久

**品饮**：  
- 小口啜饮，让酒液布满口腔  
- 分初段、中段、后段感受变化  
- 注意：甜、酸、苦、咸、鲜五味协调  
- 体会：入口、落口、回味、空杯

**评体**：  
适量饮后感受体感，优质酱酒通常醒酒较快（因人而异，请适量饮酒）。

### 8.2 适饮温度

- **最佳品饮温度**：20-25℃
- 过低：香气收敛，口感偏冷
- 过高：酒精挥发过快，香气散失

### 8.3 醒酒建议

- 开瓶后静置5-10分钟，让酒体与空气接触"呼吸"
- 老酒、高度酒可适当延长醒酒时间
- 不建议过度醒酒，以免香气散失

### 8.4 配餐建议

| 场景 | 推荐产品 | 配餐 |
|------|----------|------|
| 日常小酌 | 石掌柜·真坤沙 | 花生米、卤味、家常菜 |
| 朋友聚会 | 石荣霄青云志 | 烧烤、火锅、湘菜 |
| 商务宴请 | 石荣霄荣耀经典 | 粤菜、淮扬菜、海鲜 |

| 文化礼品 | 周秉衡烧坊典藏 | 搭配文化主题包装 |

### 8.5 饮酒健康提示

- 成年人适量饮酒，建议每日纯酒精摄入不超过25克（约53度白酒50ml）

## 十五、酱酒行业深度解读（经销商进阶）

### 15.1 茅台镇产区知识

**核心产区7.5平方公里：**  
茅台镇位于赤水河下游，东经106°、北纬27°，独特的小盆地气候形成不可复制的酿酒微生物环境。2004年贵州省政府划定7.5平方公里为酱酒核心产区，石荣霄酒庄位于此范围内。

**赤水河水系：**  
赤水河全长436.5公里，流域面积2.04万平方公里，水质pH值偏酸性，富含多种微量元素，是酱酒酿造的理想用水。每年端午节前后河水因两岸植被冲刷呈现赤红色，故称"赤水河"。

**红缨子高粱：**  
茅台镇本地糯高粱，粒小皮厚、支链淀粉含量高，能经受多次蒸煮发酵。只有这种高粱才能支撑完整的12987坤沙工艺。

### 15.2 酱酒勾调艺术

勾调不是"掺假"，而是酱酒生产的**核心工艺环节**：

- **目的**：使酒体风格统一、口感协调、批次稳定
- **原料**：不同轮次、不同年份、不同等级的基酒
- **比例**：通常需要上百种基酒小样试验
- **老酒作用**：增加陈香、使酒体柔和、提升层次感
- **大师价值**：一级酒体设计师（如刘阳均）的勾调技艺是品质保障

石荣霄每瓶酒均加入窖藏数十年老酒勾调，这是区别于普通酱酒的关键。

### 15.3 酱酒收藏投资价值

**适合收藏的酱酒特征：**
- 大曲坤沙工艺
- 知名品牌或历史底蕴
- 限量或纪念款
- 完整包装、品相良好
- 正规渠道、可溯源

**石荣霄收藏推荐：**
- 荣耀1879纪念款 — 1879创始纪念
- 10年/15年陈酿 — 真实年份标注
- 巴拿马金奖复刻版 — 历史意义

**注意事项：**
- 酱酒收藏需长期持有，3-5年起
- 避免购买来源不明的产品
- 注意保存条件
- 投资有风险，收藏需理性

### 15.4 2025-2026酱酒消费趋势

1. **理性消费**：消费者不再盲目追高价，更关注品质与性价比
2. **口粮化**：100-300元成为酱酒消费主力价格带

4. **年轻化**：低度、果味、时尚包装吸引Z世代
